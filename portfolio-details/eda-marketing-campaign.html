Project information

  • Category: Exploratory Data Analysis (EDA)
  • Project date: 2024
  • Project URL: Kaggle | Github

The project is designed to uncover valuable insights from a company's marketing campaign data using statistical and visual analysis techniques. This project aims to provide a deep understanding of campaign performance, customer behavior, and potential areas for optimization. The EDA process will begin by collecting and cleaning the marketing campaign dataset, which may include variables such as customer demographics, campaign types, engagement metrics, and conversion rates. Once the data is prepared, the project will involve a systematic exploration of the dataset using various statistical methods and data visualization tools.
